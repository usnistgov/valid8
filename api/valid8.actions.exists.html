valid8.actions.exists module¶

valid8.actions.exists.exists(_boolean, context)[source]¶

Action exists will check that every item in the context does exist.

Parameters:
  • _boolean – Placeholder for now. Holds the True value from the exists: True statement
  • context (list) – the filtered context, e.g. all files to apply this rule to.
Returns: tuple
(True, []) if all files were found. (False, error_list) if not.

Possible errors: NoSuchFile for each item in context not found.
