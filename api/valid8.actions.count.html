valid8.actions.count module¶

valid8.actions.count.count(n, context)[source]¶

Action count checks that the context created by the filters respects the cardinality n.

Parameters:
  • n – Expected number of files. Can be an integer such as “1” or “3” expecting that exact number of files, or a string that is an integer followed by a ‘+’, such as “2+”, expecting at least that number of files.
  • context (list) – the filtered context, e.g. all files to apply this rule to.
Possible errors:
single IncorrectFileCount single InsufficientFileCount

Example

  • If the context matches the cardinality n, (e.g. 1 file for n==1 or 7 n==‘5+’):
    returns (True, [])
  • If the context does not match the cardinality n, (e.g. 2 files for n==1):
    returns (False, [IncorrectFileCount()])
Returns:(output boolean, error list)
Return type:tuple
Raises:UnknownArgument – is raised when “n” is formatted incorrectly
