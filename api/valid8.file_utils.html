valid8.file_utils module¶

valid8.file_utils.filepath_exists(filepath)[source]¶

Checks for file existence. Works for any file type, including directories. Uses pathlib library.

Parameters:filepath (str) – File path to test for existence
Returns:True if the file exists
Return type:bool
valid8.file_utils.pattern_exists(pattern)[source]¶

Uses glob to find all EXISTING files that match pattern.

Parameters:pattern (str or Path-like) – file path, can include wildcards
Returns:existing pattern matches.
Return type:list
valid8.file_utils.pattern_matches(pattern)[source]¶

Uses glob to find all the file paths that pattern may match. DOES NOT garantee file existence.

If not using regex/wildcards, will return [pattern], whether or not pattern exists.

Parameters:pattern (str or Path-like) – file path, can include wildcards
Returns:file paths that match the pattern
Return type:list
